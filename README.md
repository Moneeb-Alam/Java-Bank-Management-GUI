# Bank Management System (Java Swing)

A desktop GUI application built using Core Java and Java Swing. This project currently uses File Handling (TXT files) to store user data and transaction records safely.

##  Core Features
* **Secure Authentication:** Includes 16-digit Card Number and 4-digit PIN generation and validation.
* **Account Registration:** Full signup form with dropdowns and input validation.
* **Cash Transactions:** Interactive panels for deposits and withdrawals with real-time balance calculations.
* **Data Persistence:** Uses safe file rewriting techniques to prevent data corruption or loss during operations.

##  Future Roadmap
* Migrate data storage from standard File Handling to a **MySQL Database**.
